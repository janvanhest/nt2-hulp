# Terminologie

Eenduidige begrippen voor thema, selectie en oefenen. Zie [open-vragen.md](open-vragen.md) voor vastgelegde keuzes.

| Begrip | Definitie |
|--------|-----------|
| **Thema** | Label om werkwoorden en/of zinnen te groeperen (bijv. "beweging", "les 5"). Een werkwoord of invulzin kan bij meerdere thema's horen (m:n). We gebruiken alleen "thema"; "categorie" wordt niet gebruikt. |
| **Categorie** | Wordt niet gebruikt; zie **Thema**. |
| **Oefening** | Een gegenereerde set items (vervoegingsoefening of invulzin-oefening) met bijbehorend nakijkmodel. De student maakt een oefening door items in te vullen. |
| **Selectie** | De keuze (werkwoorden, thema, eventueel expliciete zinnen) waaruit een oefening wordt samengesteld. De beheerder maakt een selectie bij het genereren van een oefening. |
| **Item** | Eén element in een oefening: een vervoeging-item (één werkwoord met alle vormen) of een invulzin-item (één zin met invulplek). |
| **Sessie** | (Optioneel, voor later) De actie van een student die "een oefening doet" (start, vult in, bekijkt nakijkmodel). Kan gebruikt worden voor voortgang/tracking. |

## Notities

- Werkwoord, invulzin, werkwoordsvorm, nakijkmodel: zie [erd.md](erd.md) en [casus.md](casus.md).
- Bij twijfel: vastleggen in [open-vragen.md](open-vragen.md) en na beslissing hier invullen.
